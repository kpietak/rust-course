## Wektory 2D

1. Zimplementuj strukturę `Vec2D` reprezentującą dwuwymiarowy wektor $v = [x, y], x, y \in \mathbb{R}$. 
1. Zdefiniuj metodę pozwalającą na czytelne wypisanie wektora na konsolę.
1. Dodaj do wektora metodę fabryczną dla wektora jednostkowego.
1. Napisz metodą sprawdzającą czy dwa wektory są równe.
1. Napisz podstawowe operatory dla działań na wektorach: dodawanie, odejmowanie, mnożenie wektora przez liczbę oraz iloczyn skalarny. Dla podstawowych operatorów wykorzystaj predefiniowane cechy (`Add`, `Sub`).
1. Napisz kod prezentujacy działania na wektorach.

```rust
use std::ops::{Add, Sub, Mul};

#[derive(Debug, Copy, Clone)]
struct Vec2D {
    x: i32,
    y: i32,
}

impl Add for Vec2D {
    type Output = Self;

    fn add(self, other : Self) -> Self {
        Vec2D{ 
            x: self.x + other.x, 
            y: self.y + other.y 
        }
    }
}

impl Sub for Vec2D {
    type Output = Self;

    fn sub(self, other : Self) -> Self {
        Vec2D {
            x : self.x - other.x,
            y : self.y - other.y
        }
    }
}

impl Mul for Vec2D {
    type Output = i32;

    fn mul(self, other : Self) -> i32 {
        self.x * other.x + self.y * other.y
    }
}


fn main() {
    let n = Vec2D{x: 1, y: 2};
    let m = Vec2D{x: 3, y: 5};

    let p = n + m;
    let p = p - n;

    println!("{:#?}", (n + m) * p);

}
```
